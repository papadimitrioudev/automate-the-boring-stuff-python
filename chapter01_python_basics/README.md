# Chapter 01 – Python Basics

This project is a beginner-level introduction to Python fundamentals,
based on Chapter 1 of *Automate the Boring Stuff with Python*.

## What this project demonstrates
- Running a Python script
- Printing output to the screen
- Getting user input
- Using variables to store data

## How it works
The program asks the user for their name and age, then prints a friendly
summary message using that information.

## Learning goal
Understand how a Python program executes line by line and how basic
input/output works.
