# Chapter 03 - Functions

## Description

This project demonstrates the fundamental concepts of functions from Chapter 03 of "Automate the Boring Stuff with Python". It includes a simple temperature converter program that uses functions to organize code and handle conversions between Celsius and Fahrenheit.

## Concepts Covered

* Defining functions with def statements
* Parameters and arguments
* Return values
* None value
* Local and global scope
* Exception handling with try/except
* Code organization and reusability

## Project: Temperature Converter

A simple interactive program that converts temperatures between Celsius and Fahrenheit using functions. The program validates user input and handles errors gracefully.

## Features

* Convert Celsius to Fahrenheit
* Convert Fahrenheit to Celsius
* Input validation with error handling
* Clear function organization
* Option to perform multiple conversions

## How to Run

### Windows Command Prompt

cd chapter03_functions
python src/main.py

### PyCharm

1. Open the project folder in PyCharm
2. Right-click on src/main.py
3. Select "Run 'main'"

## Learning Outcomes

* Creating and calling functions
* Passing data to functions using parameters
* Returning values from functions
* Understanding variable scope
* Handling user input errors
* Organizing code with functions for better readability

## Notes

This is a beginner-level project focused on function fundamentals. The code demonstrates how functions make code more organized, reusable, and easier to maintain.
